# Assets Directory

Place your app assets here:

## Required Files

- `icon.png` - App icon (1024x1024 recommended)
- `splash-icon.png` - Splash screen icon (512x512 recommended)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024)
- `favicon.png` - Web favicon (48x48)
- `notification-icon.png` - Push notification icon (96x96, Android)

## Icon Generation

You can use Expo's icon generator:
```bash
npx expo-optimize
```

Or online tools like:
- https://icon.kitchen
- https://appicon.co

## Recommendations

- Use PNG format with transparency
- App icon should have rounded corners built-in for iOS
- Splash icon should be centered with padding
- Keep notification icon simple and single-color
